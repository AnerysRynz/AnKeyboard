# AnKeyboard

AnKeyboard is a smart keyboard for Android that learns from user typing habits. This keyboard provides modern features such as autocorrect, emoji picker, gesture typing, dark mode, and text selection.

## Main Features

- **Auto Learning**: Keyboard learns from words you frequently type
- **Autocorrect**: Automatic correction suggestions with highlight
- **Emoji Picker**: Complete emoji collection
- **Gesture Typing**: Swipe for various functions (delete, space, hide, emoji)
- **Dark Mode**: Automatic dark theme support
- **Text Selection**: Complete editing toolbar like GBoard
- **Material Design**: Modern UI with animations and effects

## Installation

1. Clone this repository
2. Open with Android Studio
3. Build and install to device

## Usage

1. Install the application
2. Open Settings > Language & Input > Virtual Keyboard
3. Enable AnKeyboard
4. Select AnKeyboard as default keyboard
5. Start typing!
